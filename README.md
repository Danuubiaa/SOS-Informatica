# SOS Informática

Aplicação web para auxílio no diagnóstico de problemas básicos de informática. O sistema orienta o usuário por meio de perguntas objetivas, indicando possíveis causas e soluções sem a necessidade de conhecimento técnico prévio.

---

## Funcionalidades

- **Computador sem vídeo** — verificação de cabos, monitor e placa de vídeo
- **Internet não funciona** — diagnóstico de conexão, roteador e configurações de rede
- **Computador lento** — identificação de causas comuns de lentidão e sugestões de melhoria

---

| Tecnologia | Finalidade |

## Como Executar

```bash
git clone https://github.com/Danuubiaa/SOS-Informatica.git
cd SOS-Informatica
```

Abra o arquivo `index.html` no navegador. Nenhuma instalação adicional é necessária.
